# Manual Scripts

These scripts are intentionally kept out of `tests/` because they are exploratory, slow, or require external services.

- `run_core_algorithm_benchmark.py`: local Stage 1/2 benchmark over `tests/data`
- `run_coverage_audit.py`: reconstruction coverage audit over `tests/data`
- `run_zero_feature_demo.py`: semantic-only zero-feature end-to-end demo (requires LLM access)
